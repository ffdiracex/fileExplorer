File explorer in python with a simple gui.
built with focus on type safety and following oop standards.

--- WHY ---
This is a school project from the beginning, and serves to be a start of a larger project in the future.
We need to prove that python is capable of many things, including file exploration.
Since the author uses the terminal emulator kitty on a daily basis, which is written in python,
we need to also use python. It just makes sense, note that we also use python in school and is fitting for the project thesis.



---- prerequisites ----
Python 3.13 or newer. (try python 2.* and report the results. add @user9993310 on discord)
Tested on both linux and windows.

---- install ----
1. python -m venv env
2. source env/bin/activate (linux), .\env\Scripts\activate.bat (windows)
3. pip install -r requirements.txt
4. python main.py

---- debugging ----
the file explorer can be run with debug mode ON,
with this mode toggled you will get messages in 
the terminal in which you are executing the python
file in, main.py. If using an IDE, upon running the
file, it will open up a terminal window and you will
be met with debug messages. You can decide the level
of verbosity of these, and their details.
